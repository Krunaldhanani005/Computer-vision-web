Capabilities

Advanced Computer Vision AI

Production-grade models working together to deliver intelligent, real-time perception across diverse environments.

Face Recognition
DNN-powered face detection with 128-dimensional encoding matching. Supports known, blacklist, and unknown person classification with per-frame identity tracking.
Object Detection
YOLOv8 real-time object detection scoped to bottles, chairs, watches, and mobile phones with bounding boxes and confidence scores.
Restricted Area Security
YOLO person detection + face encoding matching against a registered persons database. Logs intrusion alerts with timestamped snapshots in MongoDB.
Real-time Streaming
MJPEG multi-stream pipeline with frame-skip optimisation delivering 15–30 FPS. Supports webcam and authenticated RTSP / HTTP CCTV sources.
Zone Definition
Draw custom monitoring zones directly on the video feed. Zones are stored in MongoDB and the model only triggers alerts for detections inside the defined boundary.
MongoDB Persistence
All face encodings, monitoring zones, known persons, and intrusion alert records are stored persistently in MongoDB with full query and reload support.
Vehicle Detection
Upload a video file for YOLOv8-based vehicle and license plate detection. Processes and streams annotated frames with bounding boxes and OCR plate text.
